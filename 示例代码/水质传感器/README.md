# 项目综合介绍

本智能防控福寿螺作品是一套集成了监测、定位、数据管理与清除执行的全流程解决方案，通过多模块协同运作实现对福寿螺的精准防控，适用于农业生态监测与有害生物防治场景。

## 核心功能与技术架构
- **智能监测体系**：基于Gemini355深度相机与YOLO11模型，实现福寿螺的实时识别与三维定位，精准获取目标坐标信息；同时通过pH、温度、TDS、浊度等水质传感器，全面监测水域环境参数，为福寿螺生长环境分析提供数据支撑。
- **数据管理与可视化平台**：采用Vue.js+Flask技术栈构建前端后端系统，集成高德地图API实现福寿螺分布位置、密度热力图及作业船轨迹的可视化展示，支持异常聚集区域警报、水质参数实时显示及数据趋势统计分析，为防控决策提供直观依据。
- **执行控制模块**：配备基于鸿蒙OS（LiteOS内核）的3自由度+1夹爪机械臂，可通过串口指令实现精准运动控制，结合定位系统数据完成对福寿螺的定点清除操作。
- **数据传输与集成**：通过ws63系列单片机采集水质数据，经4G模块以JSON格式上报至华为云平台，实现远程监控与数据汇聚，同时各模块间通过串口、I2C等接口完成本地数据交互与协同工作。

本项目包含多个子项目，涉及水上福寿螺监测、福寿螺定位、传感器数据显示、机械臂控制以及水质数据上报等多个方面，以下是各子项目的简要介绍：

## website
这是一个基于Vue.js和Flask的系统，集成了地图显示、异常警报、水质监测等功能模块，旨在提供直观的福寿螺分布监测和清除作业管理解决方案。

## detect
本项目基于Gemini355深度相机实现福寿螺的单一目标定位，集成了目标识别与三维测距功能。

## sensor
该项目代码是一个针对ws63系列单片机的综合传感器演示例程，通过I2C总线与多个传感器和一个SSD1306 OLED显示屏进行通信。

## ArmControl
本项目是一个为基于鸿蒙OS（LiteOS内核）的BearPi - Pico开发板设计的3自由度 + 1夹爪机械臂的完整控制程序。

## mqttconnect
本项目代码专为ws63系列单片机设计，采集多种水质传感器的数据，然后通过UART串口将这些数据发送给外接的4G通信模块，最终上报到华为云物联网（IoTDA）平台。
