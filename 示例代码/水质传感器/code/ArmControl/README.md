# 基于 PServo.c 的机械臂控制器

## 概述

本项目是一个为基于鸿蒙OS（LiteOS内核）的 BearPi-Pico 开发板设计的 3 自由度 + 1 夹爪机械臂的完整控制程序。它使用 PCA9685 16通道 I2C 舵机驱动板来控制舵机，并通过串口（UART）接收指令，实现对机械臂的精确控制。

程序的核心是运动学解算，包括：
*   **逆向运动学 (Inverse Kinematics)**: 将目标笛卡尔坐标 (X, Y, Z) 转换为各个关节舵机需要转动的角度。
*   **正向运动学 (Forward Kinematics)**: 根据当前各关节舵机的角度，计算出末端（夹爪）的 (X, Y, Z) 坐标。

## 主要功能

*   **坐标控制**: 通过发送 `X,Y,Z` 坐标指令，直接控制机械臂末端移动到空间中的指定位置。
*   **单轴控制**: 支持独立移动 X, Y, Z 轴。
*   **平滑运动**: 采用多轴线性插值算法，确保机械臂在移动过程中动作连贯、平滑。
*   **UART 指令接口**: 提供了一套简单易用的文本指令集，方便通过串口调试助手或上位机进行控制。
*   **舵机校准**: 支持为每个舵机设置角度偏移量，以补偿物理安装误差。
*   **物理限位**: 可为每个舵机配置最小和最大转动角度，防止机械结构损坏。
*   **实时任务**: 控制逻辑运行在 LiteOS 的一个独立任务中，保证了系统的实时响应。
*   **状态查询**: 可以查询机械臂当前的坐标和各舵机的角度。

## 硬件连接

1.  **主控板**: BearPi-Pico (或兼容的 WS63 系列开发板)。
2.  **舵机驱动**: PCA9685 I2C 舵机驱动板。
3.  **舵机**:
    *   **基座 (Base)**: 连接到 PCA9685 的 `Channel 0`。
    *   **肘部 (Elbow)**: 连接到 PCA9685 的 `Channel 1`。
    *   **肩部 (Shoulder)**: 连接到 PCA9685 的 `Channel 2`。
    *   **夹爪 (Gripper)**: 连接到 PCA9685 的 `Channel 3`。
4.  **串口连接**:
    *   连接到RDK X5开发板。
    *   **TX Pin**: `GPIO 8`
    *   **RX Pin**: `GPIO 7`
    *   **波特率**: `115200`
    *   **数据位**: `8`, **停止位**: `1`, **校验位**: `None`

## 软件配置

所有关键参数都在 `application/samples/ArmControl/src/PServo.c` 文件的开头部分通过宏定义进行配置：

*   `ARM_L1`, `ARM_L2`: 机械臂的大臂和小臂长度，用于运动学计算。
*   `BASE_SERVO_CHANNEL`, `ELBOW_SERVO_CHANNEL`, 等: 定义每个舵机连接到 PCA9685 的通道。
*   `BASE_ANGLE_MIN`/`MAX`, `SHOULDER_ANGLE_MIN`/`MAX`, 等: 定义每个舵机物理上的最小和最大角度，防止过载。
*   `BASE_SERVO_OFFSET`, `SHOULDER_SERVO_OFFSET`, `ELBOW_SERVO_OFFSET`: 舵机校准偏移量。用于修正理论角度和实际安装角度之间的差异。计算公式为 `实际角度 - 理论角度`。
*   `BASE_GEAR_RATIO`: 基座舵机与底盘之间的齿轮比。
*   `SMOOTH_MOTION_ENABLED`, `MOTION_STEP_DELAY_MS`, `MOTION_STEP_SIZE`: 平滑运动的参数，可调整移动速度和流畅度。

## 使用方法 (UART 指令集)

通过串口调试助手向开发板发送以下指令（指令以回车换行符结束）：

| 指令格式           | 功能描述                                         | 示例          |
| :------------- | :------------------------------------------- | :---------- |
| `<x>,<y>,<z>`  | **绝对坐标移动**: 将机械臂末端移动到指定的 (X, Y, Z) 坐标。       | `150,100,0` |
| `X<value>`     | **移动X轴**: 保持当前 Y 和 Z 坐标不变，移动 X 轴到 `<value>`。 | `X180`      |
| `Y<value>`     | **移动Y轴**: 保持当前 X 和 Z 坐标不变，移动 Y 轴到 `<value>`。 | `Y120`      |
| `Z<value>`     | **移动Z轴**: 保持当前 X 和 Y 坐标不变，移动 Z 轴到 `<value>`。 | `Z50`       |
| `S <ch> <ang>` | **直接控制舵机**: 将通道 `<ch>` 的舵机转动到角度 `<ang>`。     | `S2 90`     |
| `GRAB`         | **夹紧夹爪**: 控制夹爪闭合。                            | `GRAB`      |
| `RELEASE`      | **松开夹爪**: 控制夹爪张开。                            | `RELEASE`   |
| `POS`          | **查询位置**: 在串口输出机械臂当前的 (X,Y,Z) 坐标和各舵机角度。      | `POS`       |

### 坐标系说明

*   **X轴**: 向前为正方向。
*   **Y轴**: 向上为正方向。
*   **Z轴**: 向左为正方向。
*   **原点**:  机械臂底座中心点
