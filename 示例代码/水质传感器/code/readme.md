# 项目综合介绍

本项目包含多个子项目，涉及水上福寿螺监测、福寿螺定位、传感器数据显示、机械臂控制以及水质数据上报等多个方面，以下是各子项目的简要介绍：

## website
这是一个基于Vue.js和Flask的系统，集成了地图显示、异常警报、水质监测等功能模块，旨在提供直观的福寿螺分布监测和清除作业管理解决方案。

## detect
本项目基于Gemini355深度相机实现福寿螺的单一目标定位，集成了目标识别与三维测距功能。

## sensor
该项目代码是一个针对ws63系列单片机的综合传感器演示例程，通过I2C总线与多个传感器和一个SSD1306 OLED显示屏进行通信。

## ArmControl
本项目是一个为基于鸿蒙OS（LiteOS内核）的BearPi - Pico开发板设计的3自由度 + 1夹爪机械臂的完整控制程序。

## mqttconnect
本项目代码专为ws63系列单片机设计，采集多种水质传感器的数据，然后通过UART串口将这些数据发送给外接的4G通信模块，最终上报到华为云物联网（IoTDA）平台。
